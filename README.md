# infographik

Интерактивный одностраничный сайт с картой эмпатии для анализа экосистемы Yandex Go в 2024–2025 годах.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Features

- hero-блок с ключевыми метриками исследования
- интерактивные персоны Анны и Александра
- схема карты эмпатии с inline-раскрытием секторов
- отдельные блоки болей, выгод и стратегических рекомендаций
- адаптивный mobile-first интерфейс

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
